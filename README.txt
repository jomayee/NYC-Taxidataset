This directory contains five python scripts and the project paper.
Packages Used:

numpy
scipy
scikit-learn
matplotlib
bokeh
pygeogrids
seaborn
datetime
pandas
jupyter notebook

Script:
1. Processing.py -> This script is for cleaning and processing the dataset
2. merge.py -> This script is used to filter data accordingly and create new columns.
3. rbind.py -> Used to merge multiple input files into one.
4. Regression.py -> Linear Regression and Random Forest Regression models
5. Classification.py -> Logistic Regression and Decision Tree Classification models.

Visualizations are done using Bokeh plotting and Jupyter Notebook.
